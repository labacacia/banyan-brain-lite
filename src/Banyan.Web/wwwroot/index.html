Architecture

        

          Banyan is a memory node for AI agents. Memory operations go through /api/memory
          (BM25 search over SQLite + FTS5). Agents authenticate with NID certificates issued by an
          embedded NPS-NIP CA — humans use OIDC/JWT (run banyan login). All keys never leave
          this host except the public halves.
