# Banyan Brain Lite 1.1.0 Release Notes

Banyan Brain Lite 1.1.0 is the post-Wave GA release. It keeps the single-node, SQLite-backed Lite boundary from 1.0.0 and adds the knowledge-pack, observability, and dependency-hardening work that landed after the first stable cut.

## Release scope

1.1.0 is the **Lite** edition only.

It remains intended for:

- local agent memory;
- offline-first memory nodes;
- small single-node deployments;
- demos and integration testing;
- embedded agent runtimes that need SQLite-backed durable memory.

Multi-tenant fleet topology, Ent-style policy governance, external object storage, and cloud coordination remain Pro/Ent responsibilities.

## Highlights

### Knowledge packs

- `.banyanpack` v2 manifest and migration support.
- Ed25519 pack signing and verification.
- Pack mount trust through the NID/CA chain.
- In-pack embeddings and cosine recall so Lite can search mounted packs without importing every record into the live memory store.
- Pack version management: pin, upgrade, rollback, and active-version switching.

### Retrieval and packaging

- Lite recall can merge live memory hits with mounted knowledge-pack vector hits.
- `Banyan.Embedders` and `Banyan.Embedders.Onnx` are split so the CLI and library packages can stay slim unless ONNX support is needed.
- ONNX project files are restored in the public distribution so fresh clones build cleanly.

### Observability and audit

- Lite OpenTelemetry foundation.
- Memory-operation metrics for local operators.
- Tamper-evident audit records for write, update, and forget operations.

### Dependency hardening

- NPS packages aligned to `1.0.0-alpha.12`.
- `MessagePack` pinned to `3.1.7` where NPS dependencies would otherwise restore vulnerable 3.0.x versions.
- `SQLitePCLRaw.bundle_e_sqlite3` pinned to `3.0.3` to avoid the vulnerable bundled SQLite native library.

## Install

```bash
dotnet tool install -g Banyan.Cli --version 1.1.0
```

## Upgrade

```bash
dotnet tool update -g Banyan.Cli --version 1.1.0
```

After upgrading, verify the tool and initialize any missing local assets:

```bash
banyan --version
banyan embedder download
```

Back up the SQLite data directory before upgrading production Lite nodes.

## Compatibility

- Target framework: `net10.0`.
- Package version: `1.1.0`.
- Storage: SQLite.
- License: Apache-2.0.
